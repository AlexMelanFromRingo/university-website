Про мене

Це Я (●'◡'●) - Починаючий розробник

Alex Melan

Ще з дитинства мені подобається комп'ютерна тематика. З часом, все більше мене поглинало ковиряння у комп'ютерних залізяках. У школі я почав вивчати програмування для саморозвитку, бо мені було цікаво почати писати якісь свої програми, ботів, скрипти, макроси й так далі. Далі мене почала цікавити тематика інформаційної безпеки, тож я почав вивчати основи криптології, вразливостей комп'ютерних систем та мереж, різні протоколи обміну даних й так далі. На даний момент, я вивчаю такі мови програмування, як:

  1. Python (боти, нейромережі, скрипти, автоматизація, парсери й так далі).
  2. C++ (Швидкі розрахунки).
  3. Java (Створення плагінів та модифікацій для гри, швидкі розрахунки й зручна робота з багатоядерними процесорами).
  4. Processing (Програмування мікроконтроллерів накшталт Arduino, ESP8266, тощо).
  5. Lua (для розробки маленьких вбудованих програм).
  6. GLSL для програм на GPU.
  7. Також трохи займаюсь моделюванням у Blender.

Цікавлюсь роботую hash-функцій, алгоритмами роботи протоколу WPA-2, WPS, децентралізованими розрахунками, криптологією, нейромережами й новинами у сфері захисту інформації.
