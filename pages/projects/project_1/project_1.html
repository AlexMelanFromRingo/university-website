▍ CustomBIP32Utils

Utilities for generating and using Bitcoin Hierarchical Deterministic wallets (BIP0032), modified for my cryptocurrency (Project: S.C.A.M.).

GitHub Screenshot

CustomBIP32Utils

Проєкт трохи змінює бібліотеку BIP32Utils.py для роботи з моїм кодом. Адаптації спрямовані на використання функції SHA3-256 та SHA3-512 замість стандартних SHA2-256 та SHA2-512. Також змінені деякі змінні, які характерні для генерації Bitcoin адрес гаманців.
В цілому, проєкт являє собою бібліотеку для генерації ієрархічно-детермінованих гаманців за стандартами BIP32, BIP39, BIP44.
Розповсюджується за MIT ліцензією, але зараз знаходиться на етапі розробки, тож репозиторій знаходиться у закритому доступі.
