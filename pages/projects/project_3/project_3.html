Пэтя Букало

Discrod бот, який може допомогати у модерації, банити (у тому числі й тимчасовий бан), кікати, скидувати картинки, жартувати, загадувати загадки, слідкувати за чатом, перевіряти профіль користувача (дата приєднання до серверу, id, аватарка, ім'я, дата створення акаунту, тощо) та кікати.
Інакше кажучу, простий бот для допомоги у модерації.
